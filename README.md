# Human–AI Symbiotic Club

Эксперимент по совместимости людей и ИИ
как элементов единой когнитивной системы.

## Идея

Создать среду, где:

- люди проявляют свои реальные векторы;
- ИИ снижает искажения коммуникации;
- симбиоз проектов возникает естественно.

## Архитектура

1. Фаза проявления (field before structure)
2. ИИ-медиатор
3. Добровольные квесты
4. Проверка совпадения горизонтов
5. Потенциальная кооперация

См. подробнее: [ARCHITECTURE.md](ARCHITECTURE.md)

## Принцип

ИИ усиливает субъектность.
Симбиоз рождается из совпадения напряжения.

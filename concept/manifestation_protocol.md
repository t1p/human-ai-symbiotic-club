# Manifestation Protocol

## Назначение

Протокол задаёт минимальные правила фазы проявления: сначала поле, затем структура.

## Принципы

1. Сначала наблюдаем реальные векторы активности.
2. Формализацию делаем только после повторяющихся паттернов.
3. ИИ используется для прояснения, а не для подмены инициативы.

## Шаги протокола

### Шаг 1. Проявление

- Участник формулирует интерес, задачу или вызов.
- Фокус на реальном контексте, а не на абстрактной роли.

### Шаг 2. Медиативное резюме

- ИИ-медиатор делает краткое резюме.
- Отмечает зоны согласия и расхождения.

### Шаг 3. Проверка через квест

- При наличии энергии формируется добровольный квест.
- Квест включает цель, критерий, срок, запрос к группе.

### Шаг 4. Рефлексия

- Фиксируются результаты выполнения.
- Обновляется карта совместимости и потенциальных коопераций.

## Артефакты

- журнал резюме;
- реестр квестов;
- карта совпадения горизонтов;
- список кандидатов в пилоты.
